## 📊 技术细节

### Mesh旋转

每个mesh文件使用trimesh库绕Y轴旋转90度：

```python
rotation_matrix = trimesh.transformations.rotation_matrix(
    np.radians(90), [0, 1, 0])
mesh.apply_transform(rotation_matrix)
```

### Joint Axis变化

| 关节 | 原来 | 现在 | 效果 |
|------|------|------|------|
| j11  | axis="-1 0 0" | axis="0 0 -1" | Z轴→X轴 |
| j111 | axis="-1 0 0" | axis="-1 0 0" | Y轴（不变） |
| j1111| axis="-1 0 0" | axis="-1 0 0" | Y轴（不变） |

### 关节链结构

```
base_link
  └─ j1 (滑轨joint, X轴滑动)
      └─ l1 (滑轨link, mesh已旋转)
          └─ j11 (髋关节, X轴旋转✅, mesh已旋转)
              └─ l11 (髋关节link)
                  └─ j111 (大腿joint, Y轴旋转)
                      └─ l111 (大腿link)
                          └─ j1111 (小腿joint, Y轴旋转)
                              └─ l1111 (小腿link)
```
