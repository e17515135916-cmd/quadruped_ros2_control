## 💡 Blender操作回顾

### 导入两个STL文件：
```bash
blender l1_upper.STL l1_lower.STL
```

### 旋转对象：
- 选择对象
- `R` + `Z` + `90` = 绕Z轴旋转90度
- 或按 `N` 在侧边栏精确输入角度

### 合并对象：
- `Shift` + 点击选择两个对象
- `Ctrl + J` 合并

### 合并重复顶点：
- `Tab` 进入编辑模式
- `A` 全选
- `M` → "按距离"
- `Tab` 退出编辑模式

### 导出：
- `File` → `Export` → `STL (.stl)`
