# 舵机旋转90度与支架改造指南

## 🎯 目标

保持蓝色舵机与白色支架的相对位置关系不变，将舵机沿Y轴旋转90度，并修改白色支架mesh，将垂直侧壁改为水平托板来接住旋转后的舵机卡扣。

---

## 📋 任务分解

### 阶段1：理解当前结构
1. 蓝色舵机（可能是l1或l2部件）
2. 白色支架（可能是l11、l21等部件）
3. 当前：支架垂直夹紧舵机两侧
4. 目标：支架水平托住舵机卡扣

### 阶段2：舵机旋转
- 以舵机几何中心为轴
- 绕Y轴旋转90度
- 保持与支架的相对位置

### 阶段3：支架改造
- 删除原有的垂直侧壁
- 添加水平托板
- 填补几何空隙

---

## 🔧 方法1：在Blender中手动操作（推荐）

### 步骤1：确定部件

首先需要确认哪个是舵机，哪个是支架：

```bash
# 查看所有mesh文件
ls -lh src/dog2_description/meshes/*.STL
```

常见的组合：
- **舵机**：l1.STL, l2.STL（蓝色部件）
- **支架**：l11.STL, l21.STL（白色部件）

### 步骤2：在Blender中打开两个部件

```bash
# 同时打开舵机和支架
blender src/dog2_description/meshes/l1.STL \
        src/dog2_description/meshes/l11.STL
```

### 步骤3：旋转舵机

**在Blender中：**

1. **选择舵机对象**（l1）
2. **设置旋转中心为几何中心**：
   - 按 `Shift + S` → "Cursor to Selected"（光标到选中对象）
   - 或者：`Object` → `Set Origin` → `Origin to Geometry`

3. **旋转90度**：
   - 按 `R` + `Y` + `90` + `Enter`
   - 或在侧边栏（按`N`）的Transform中输入：
     - Rotation Y: 90°

4. **检查位置**：
   - 确保舵机与支架的相对位置没有改变
   - 如果位置偏移，使用 `G` 键移动调整

### 步骤4：分析支架需要修改的部分

**观察旋转后的状态：**

1. 按 `Alt + Z` 开启X-Ray模式（透视）
2. 从不同角度查看（小键盘 `1`, `3`, `7`）
3. 找到：
   - 原有的垂直侧壁位置
   - 舵机卡扣的新位置
   - 需要填补的空隙

### 步骤5：修改支架mesh

**方案A：删除垂直侧壁，添加水平托板**

1. **选择支架对象**（l11）
2. **进入编辑模式**：按 `Tab`
3. **选择要删除的垂直侧壁**：
   - 按 `3` 切换到面选择模式
   - 按 `Alt + Z` 开启X-Ray模式
   - 框选或点选垂直侧壁的面
   - 按 `X` → `Faces` 删除

4. **添加水平托板**：
   - 按 `Shift + A` → `Mesh` → `Plane`（添加平面）
   - 按 `S` 缩放到合适大小
   - 按 `G` + `Z` 移动到舵机卡扣下方
   - 按 `E` 挤出，增加厚度

5. **合并到支架**：
   - 按 `Tab` 退出编辑模式
   - 选择托板和支架（`Shift + 左键`）
   - 按 `Ctrl + J` 合并

6. **填补空隙**：
   - 进入编辑模式 `Tab`
   - 选择边缘的边（按 `2` 切换到边选择模式）
   - 按 `F` 填充面
   - 或使用 `E` 挤出来填补

**方案B：使用修改器重塑支架**

1. **选择支架对象**
2. **添加修改器**：
   - 右侧面板 → `Modifiers` → `Add Modifier`
   - 选择 `Boolean` 或 `Solidify`

3. **手动编辑顶点**：
   - 进入编辑模式 `Tab`
   - 按 `1` 切换到顶点选择模式
   - 选择需要移动的顶点
   - 按 `G` + `Z` 移动到新位置

### 步骤6：检查和优化

1. **检查mesh完整性**：
   - `Select` → `Select All by Trait` → `Non Manifold`
   - 如果有非流形边，需要修复

2. **重新计算法向量**：
   - 全选 `A`
   - `Mesh` → `Normals` → `Recalculate Outside`
   - 或按 `Shift + N`

3. **合并重复顶点**：
   - 全选 `A`
   - 按 `M` → "按距离"

### 步骤7：导出修改后的mesh

1. **导出舵机**：
   - 选择舵机对象
   - `File` → `Export` → `STL (.stl)`
   - 勾选 "Selection Only"
   - 保存为 `l1_rotated.stl`

2. **导出支架**：
   - 选择支架对象
   - `File` → `Export` → `STL (.stl)`
   - 勾选 "Selection Only"
   - 保存为 `l11_modified.stl`

## 🤖 方法2：使用Python脚本辅助

如果需要精确控制，可以使用Python脚本：

```python
import bpy
import bmesh
from mathutils import Matrix, Vector
import math

# 清空场景
bpy.ops.object.select_all(action='SELECT')
bpy.ops.object.delete()

# 导入舵机和支架
bpy.ops.import_mesh.stl(filepath="src/dog2_description/meshes/l1.STL")
servo = bpy.context.selected_objects[0]
servo.name = "servo"

bpy.ops.import_mesh.stl(filepath="src/dog2_description/meshes/l11.STL")
bracket = bpy.context.selected_objects[0]
bracket.name = "bracket"

# 旋转舵机90度（绕Y轴）
servo.rotation_euler[1] = math.radians(90)

# 应用旋转
bpy.context.view_layer.objects.active = servo
bpy.ops.object.transform_apply(location=False, rotation=True, scale=False)

# 导出
bpy.ops.export_mesh.stl(filepath="l1_rotated.stl", use_selection=True)

print("✓ 舵机已旋转并导出")
```

## 📐 关键尺寸参考

在修改支架时，需要考虑：

1. **舵机卡扣尺寸**：
   - 通常宽度：2-3mm
   - 长度：根据舵机型号

2. **托板厚度**：
   - 建议：2-5mm
   - 确保足够强度

3. **间隙**：
   - 舵机与托板：0.5-1mm
   - 避免过紧或过松

## ⚠️ 注意事项

### 1. 保持相对位置
- 旋转舵机时，确保旋转中心正确
- 使用 `Object` → `Set Origin` → `Origin to Geometry`

### 2. 检查碰撞
- 旋转后检查是否有部件重叠
- 使用X-Ray模式（`Alt + Z`）查看

### 3. 保持mesh质量
- 避免产生非流形边
- 重新计算法向量
- 合并重复顶点

## 🎨 Blender快捷键速查

### 基本操作
- `Tab` = 切换编辑模式/对象模式
- `1/2/3` = 顶点/边/面选择模式
- `A` = 全选
- `Alt + A` = 取消全选
- `X` = 删除

### 变换操作
- `G` = 移动（Grab）
- `R` = 旋转（Rotate）
- `S` = 缩放（Scale）
- `G/R/S` + `X/Y/Z` = 沿指定轴操作

### 建模操作
- `E` = 挤出（Extrude）
- `F` = 填充面
- `Ctrl + R` = 循环切割
- `K` = 刀切工具
- `M` = 合并顶点

### 视图操作
- `Alt + Z` = X-Ray透视模式
- `小键盘 1/3/7` = 前/侧/顶视图
- `N` = 打开/关闭侧边栏
