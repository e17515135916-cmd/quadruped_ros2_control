# 关节配置改造：ZYY → XYY

## 🎯 目标

将Dog2机器人的髋关节配置从**ZYY**改为**XYY**：
- **原来**：第一个关节绕Z轴旋转
- **改造后**：第一个关节绕X轴旋转

## 📋 需要修改的内容

### 1. Mesh文件（旋转90度，绕Y轴）

需要旋转的文件：
- `l1.STL` - 左前腿髋关节滑轨
- `l11.STL` - 左前腿髋关节
- `l2.STL` - 右前腿髋关节滑轨
- `l21.STL` - 右前腿髋关节
- `l3.STL` - 左后腿髋关节滑轨
- `l31.STL` - 左后腿髋关节
- `l4.STL` - 右后腿髋关节滑轨
- `l41.STL` - 右后腿髋关节

以及对应的碰撞mesh。

### 2. URDF文件（修改joint axis）

需要修改的joint：
- `j11` - 左前腿髋关节
- `j21` - 右前腿髋关节
- `j31` - 左后腿髋关节
- `j41` - 右后腿髋关节

修改内容：
```xml
<!-- 原来 -->
<axis xyz="-1 0 0"/>

<!-- 改为 -->
<axis xyz="0 0 -1"/>
```

## 📐 技术细节

### Mesh旋转

每个mesh文件绕Y轴旋转90度：
- 使用trimesh库
- 旋转矩阵应用到所有顶点
- 保持文件格式不变

### Joint Axis修改

原来的配置（ZYY）：
```
j11: axis="-1 0 0" (实际是Z轴，因为mesh方向)
j111: axis="-1 0 0" (Y轴)
j1111: axis="-1 0 0" (Y轴)
```

修改后的配置（XYY）：
```
j11: axis="0 0 -1" (X轴)
j111: axis="-1 0 0" (Y轴)
j1111: axis="-1 0 0" (Y轴)
```

### 为什么这样修改？

1. **Mesh旋转90度**：改变mesh的局部坐标系方向
2. **Axis从(-1,0,0)改为(0,0,-1)**：在新的mesh坐标系下，这个轴向对应X轴
