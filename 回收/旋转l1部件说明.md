# 旋转l1部件（绕Y轴90度）

## 🎯 目标

将l1这个STL mesh文件本身绕Y轴旋转90度，同时保持：
- ✅ 整条腿的空间位置不变
- ✅ base_link的位置不变
- ✅ 所有joint的类型和位置不变
- ✅ 大腿、小腿的空间位置不变

**只有l1这个部件本身旋转了，其他所有东西都不变！**

## 🚀 快速开始

### 一键旋转（推荐）

```bash
cd ~/aperfect/carbot_ws
./旋转l1部件.sh
```

这个脚本会：
1. ✅ 自动备份原始文件
2. ✅ 在Blender中旋转l1.STL
3. ✅ 同时处理碰撞mesh
4. ✅ 提示你重新编译和查看

### 手动操作

如果你想手动操作：

## 📋 工作原理

### 修改的是什么？

**修改前：**
```
l1.STL文件内部的顶点坐标是原始方向
```

**修改后：**
```
l1.STL文件内部的顶点坐标被旋转了90度（绕Y轴）
```

### 不修改什么？

- ❌ 不修改URDF文件
- ❌ 不修改joint定义
- ❌ 不修改其他STL文件
- ❌ 不改变l1在装配体中的位置

### 为什么这样做？

因为你想要：
1. l1这个部件本身旋转
2. 但整条腿的位置保持不变
3. 所以只需要修改STL文件本身的几何形状

## 🔧 技术细节

### Blender操作

脚本在Blender中执行以下操作：

```python
# 1. 导入l1.STL
bpy.ops.import_mesh.stl(filepath="l1.STL")

# 2. 绕Y轴旋转90度
obj.rotation_euler[1] = math.radians(90)

# 3. 应用旋转（烘焙到mesh数据）
bpy.ops.object.transform_apply(rotation=True)

# 4. 导出修改后的STL
bpy.ops.export_mesh.stl(filepath="l1.STL")
```

### 旋转轴说明

- **X轴**: 红色，前后方向
- **Y轴**: 绿色，左右方向（我们绕这个轴旋转）
- **Z轴**: 蓝色，上下方向

绕Y轴旋转90度意味着：
- 原来朝前的面现在朝右
- 原来朝右的面现在朝后
- 原来朝后的面现在朝左
- 原来朝左的面现在朝前

## 📊 修改前后对比

### 修改前
```
l1部件的局部坐标系：
- X轴指向前方
- Y轴指向左侧
- Z轴指向上方
```

### 修改后
```
l1部件的局部坐标系（绕Y轴旋转90度）：
- X轴指向左侧（原来的Y轴方向）
- Y轴指向左侧（不变）
- Z轴指向后方（原来的-X轴方向）
```

## 🔄 如果需要其他旋转角度

如果你想旋转其他角度（不是90度），可以修改脚本：

编辑 `scripts/rotate_l1_mesh.py`，找到这一行：
```python
obj.rotation_euler[1] = math.radians(90)  # 90度
```

改为你想要的角度：
```python
obj.rotation_euler[1] = math.radians(45)   # 45度
obj.rotation_euler[1] = math.radians(180)  # 180度
```

或者绕其他轴旋转：
```python
obj.rotation_euler[0] = math.radians(90)  # 绕X轴
obj.rotation_euler[1] = math.radians(90)  # 绕Y轴
obj.rotation_euler[2] = math.radians(90)  # 绕Z轴
```
