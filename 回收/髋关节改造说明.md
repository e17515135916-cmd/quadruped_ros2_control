# 髋关节改造：从竖着到横着（ZYY → XYY）

## 🎯 改造目标

### 当前状态（ZYY配置）
- 蓝色髋关节连接件是**竖着**安装的
- 绕**Z轴**旋转（Yaw）
- 导轨、髋关节、大腿**不在同一水平线**上

### 目标状态（XYY配置）
- 蓝色髋关节连接件改为**横着**安装
- 绕**X轴**旋转（Roll）
- 导轨、髋关节、大腿**在同一水平线**上

## 📋 改造内容

### 1. 旋转Mesh文件（绕Y轴90度）

需要旋转的文件（四条腿的髋关节部件）：
- **l1.STL, l11.STL** - 左前腿
- **l2.STL, l21.STL** - 右前腿
- **l3.STL, l31.STL** - 左后腿
- **l4.STL, l41.STL** - 右后腿

以及对应的碰撞mesh。

### 2. 修改Joint Axis（从Z轴改为X轴）

修改四个髋关节的旋转轴：
- **j11** - 左前腿髋关节
- **j21** - 右前腿髋关节
- **j31** - 左后腿髋关节
- **j41** - 右后腿髋关节

```xml
<!-- 修改前 -->
<axis xyz="-1 0 0"/>  <!-- 实际是Z轴，因为mesh方向 -->

<!-- 修改后 -->
<axis xyz="0 0 -1"/>  <!-- 改为X轴 -->
```

### 3. 对齐效果

修改后：
- ✅ 导轨滑块在同一水平线
- ✅ 髋关节连接件在同一水平线
- ✅ 大腿连接点在同一水平线
- ✅ 四个部件Z轴高度一致

## 📐 技术细节

### Mesh旋转原理

```
原来（竖着）：
  Z
  |
  |___Y
 /
X

旋转90度后（横着）：
    Y
   /
  /___Z
 |
 X
```

### Joint Axis变化

| 关节 | 原来（ZYY） | 修改后（XYY） |
|------|------------|--------------|
| j11  | axis="-1 0 0" (Z轴) | axis="0 0 -1" (X轴) |
| j111 | axis="-1 0 0" (Y轴) | axis="-1 0 0" (Y轴) |
| j1111| axis="-1 0 0" (Y轴) | axis="-1 0 0" (Y轴) |

### 坐标对齐

修改后，所有髋关节的Z坐标保持一致：
- 导轨滑块：Z = 0.2649
- 髋关节：Z = 0.2649 + 0.055 = 0.3199
- 大腿连接点：Z = 0.3199（同一高度）
