# 碰撞体参数快速参考

## 大腿碰撞体参数（已实施）

### Cylinder 几何体
- **类型**: cylinder
- **半径**: 0.072 m
- **长度**: 0.154 m

### 原点偏移
- **xyz**: 0 -0.076 0.065
- **rpy**: 0 1.5708 0

### 应用链接
- l111（腿1大腿）
- l211（腿2大腿）
- l311（腿3大腿）
- l411（腿4大腿）

### 计算依据
- 半径 = max(mesh_width, mesh_height) / 2 × 0.9
  - = 0.15997368 / 2 × 0.9 = 0.072 m
- 长度 = mesh_length × 0.85
  - = 0.18159960 × 0.85 = 0.154 m

## 足端碰撞体参数（已存在）

### Sphere 几何体
- **类型**: sphere
- **半径**: 0.02 m

### 应用链接
- l11111（腿1足端）
- l21111（腿2足端）
- l31111（腿3足端）
- l41111（腿4足端）

## 设计原则

1. **碰撞体尺寸**: 略小于实际 mesh（0.85-0.9 倍）
2. **相邻间隙**: >= 5mm（大腿-小腿）
3. **足端间隙**: >= 10mm（小腿-足端）
4. **Visual 保留**: 所有 visual 标签仍使用原始 STL mesh
