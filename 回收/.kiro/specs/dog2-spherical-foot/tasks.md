# 实施计划：球形足端

## 概述

本实施计划将DOG2机器人的足端从STL网格模型改为球形几何体，以改善接触稳定性。实施包括修改URDF/Xacro文件、更新物理参数、实现属性测试，并验证Gazebo集成。

## 任务

- [x] 1. 修改URDF足端几何体为球形
  - 在 `dog2.urdf.xacro` 的leg宏中，将足端Link的visual和collision几何体从STL mesh改为sphere
  - 设置球体半径为0.02米
  - 更新visual材质颜色为灰色 (0.5, 0.5, 0.5, 1)
  - _需求: 1.1, 1.2, 1.3_

- [x] 1.1 编写属性测试验证球形几何体

  - _需求: 1.1, 1.2, 1.3_

- [ ] 2. 更新足端惯性属性

    - 质量设置为0.05kg
    - 半径0.02m，计算得 I = 8e-6 kg·m²
    - _需求: 3.1, 3.2_

  - [ ] 2.2 更新URDF中的惯性定义

    - 设置Ixy = Ixz = Iyz = 0
