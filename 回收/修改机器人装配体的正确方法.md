## 📋 步骤2：理解机器人结构

Dog2机器人的层级结构：

```
base_link (机身)
├── l1 (左前腿 - 髋关节滑轨)
│   └── l11 (左前腿 - 髋关节)
│       └── l111 (左前腿 - 大腿)
│           └── l1111 (左前腿 - 小腿)
│               └── l11111 (左前腿 - 脚)
├── l2 (右前腿...)
├── l3 (左后腿...)
└── l4 (右后腿...)
```

每个joint定义了子link相对于父link的位置和旋转。

## 🔧 步骤3：修改URDF文件

### 3.1 打开URDF文件

```bash
cd ~/aperfect/carbot_ws
gedit src/dog2_description/urdf/dog2.urdf.xacro
```

或者用你喜欢的编辑器：
```bash
code src/dog2_description/urdf/dog2.urdf.xacro
# 或
nano src/dog2_description/urdf/dog2.urdf.xacro
```

### 3.2 找到要修改的joint

假设你想修改左前腿髋关节的位置，搜索 `j11`：

```xml
<joint name="j11" type="revolute">
    <origin rpy="0 0 1.5708" xyz="-0.016 0.0199 0.055"/>
    <parent link="l1"/>
    <child link="l11"/>
    <axis xyz="-1 0 0"/>
    <limit effort="50" lower="-2.618" upper="2.618" velocity="20"/>
</joint>
```

### 3.3 理解origin参数

- **xyz**: 位置偏移（单位：米）
  - x: 前后方向（正值向前）
  - y: 左右方向（正值向左）
  - z: 上下方向（正值向上）

- **rpy**: 旋转（单位：弧度）
  - r (roll): 绕X轴旋转
  - p (pitch): 绕Y轴旋转
  - y (yaw): 绕Z轴旋转

### 3.4 修改数值

例如，将髋关节向上移动2cm，向前移动1cm：

**原始值：**
```xml
<origin rpy="0 0 1.5708" xyz="-0.016 0.0199 0.055"/>
```

**修改后：**
```xml
<origin rpy="0 0 1.5708" xyz="-0.006 0.0199 0.075"/>
```
- x: -0.016 → -0.006 (向前1cm)
- z: 0.055 → 0.075 (向上2cm)

### 3.5 旋转修改示例

如果要绕Z轴旋转5度（0.0873弧度）：

**原始值：**
```xml
<origin rpy="0 0 1.5708" xyz="-0.016 0.0199 0.055"/>
```

**修改后：**
```xml
<origin rpy="0 0 1.6581" xyz="-0.016 0.0199 0.055"/>
```
- yaw: 1.5708 → 1.6581 (增加5度 = 0.0873弧度)

## 📐 常用修改场景

### 场景1：调整腿的安装位置

修改prismatic joint (j1, j2, j3, j4) 的xyz值：

```xml
<!-- 左前腿 -->
<joint name="j1" type="prismatic">
    <origin rpy="1.5708 0 0" xyz="1.1026 -0.80953 0.2649"/>
    ...
</joint>
```

- 修改xyz来改变腿相对于机身的位置

### 场景2：调整髋关节角度

修改hip joint (j11, j21, j31, j41) 的rpy值：

```xml
<joint name="j11" type="revolute">
    <origin rpy="0 0 1.5708" xyz="-0.016 0.0199 0.055"/>
    ...
</joint>
```

- 修改rpy来改变髋关节的初始角度

### 场景3：调整大腿长度方向

修改thigh joint (j111, j211, j311, j411) 的xyz值：

```xml
<joint name="j111" type="revolute">
    <origin rpy="1.5708 1.5708 0" xyz="-0.0233 -0.055 0.0274"/>
    ...
</joint>
```

- 修改xyz来改变大腿相对于髋关节的位置

## ⚠️ 注意事项

1. **备份原文件**
```bash
cp src/dog2_description/urdf/dog2.urdf.xacro src/dog2_description/urdf/dog2.urdf.xacro.backup
```

2. **单位**
   - 位置：米（m）
   - 角度：弧度（rad）

3. **坐标系**
   - X轴：前后（正值向前）
   - Y轴：左右（正值向左）
   - Z轴：上下（正值向上）

4. **小步修改**
   - 每次只修改一个joint
   - 修改量不要太大（建议1-2cm或5-10度）
   - 立即查看结果

5. **相对坐标**
   - joint的origin是相对于父link的
   - 修改会影响所有子link

## 🆘 常见问题

### Q: 修改后机器人看起来很奇怪？
A: 检查：
1. 是否修改了正确的joint
2. 数值是否合理（不要太大）
3. 是否正确编译了

### Q: 如何知道修改哪个joint？
A: 
- 在RViz中勾选"TF"显示坐标系
- 查看上面的层级结构图
- 每个joint连接两个link

### Q: 修改后Gazebo中机器人行为异常？
A: 如果修改了几何形状，需要同时更新：
1. 碰撞模型
2. 惯性参数

但如果只是小幅调整位置，通常不需要。

### Q: 能否撤销修改？
A: 
```bash
# 恢复备份
cp src/dog2_description/urdf/dog2.urdf.xacro.backup src/dog2_description/urdf/dog2.urdf.xacro
colcon build --packages-select dog2_description
```
