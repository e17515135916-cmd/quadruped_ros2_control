## 🚀 快速开始

```bash
cd ~/aperfect/carbot_ws

```

这个脚本提供菜单，可以：
- 在RViz中查看机器人
- 编辑URDF文件
- 重新编译
- 角度转换工具
- 备份/恢复功能

### 方法2：手动操作

#### 1. 查看机器人
```bash
cd ~/aperfect/carbot_ws
source install/setup.bash
ros2 launch dog2_description view_dog2.launch.py
```

#### 2. 编辑URDF（在另一个终端）
```bash
cd ~/aperfect/carbot_ws
gedit src/dog2_description/urdf/dog2.urdf.xacro
```

搜索你要修改的joint（如 `j11`），修改 `origin` 的值：
```xml
<origin rpy="0 0 1.5708" xyz="-0.016 0.0199 0.055"/>
```

- **xyz**: 位置（单位：米）
- **rpy**: 旋转（单位：弧度）

#### 3. 重新编译
```bash
colcon build --packages-select dog2_description
source install/setup.bash
```

#### 4. 重新启动RViz查看
```bash
ros2 launch dog2_description view_dog2.launch.py
```

## 🔧 修改示例

### 示例1：将髋关节向上移动2cm

找到joint `j11`：
```xml
<!-- 原始 -->
<origin rpy="0 0 1.5708" xyz="-0.016 0.0199 0.055"/>

<!-- 修改后（z增加0.02） -->
<origin rpy="0 0 1.5708" xyz="-0.016 0.0199 0.075"/>
```

### 示例2：旋转髋关节5度

5度 = 0.0873弧度

```xml
<!-- 原始 -->
<origin rpy="0 0 1.5708" xyz="-0.016 0.0199 0.055"/>

<!-- 修改后（yaw增加0.0873） -->
<origin rpy="0 0 1.6581" xyz="-0.016 0.0199 0.055"/>
```

## 🧮 角度转换

### 使用脚本
```bash
./查看和修改机器人.sh
# 选择选项8：角度转换工具
```

### 手动转换
```bash
# 度转弧度
python3 -c "import math; print(math.radians(5))"

# 弧度转度
python3 -c "import math; print(math.degrees(1.5708))"
```

### 常用角度
- 5° = 0.0873 弧度
- 10° = 0.1745 弧度
- 45° = 0.7854 弧度
- 90° = 1.5708 弧度
